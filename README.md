 
# 🚀 Запуск
Подключиться к серверу (VDS/VPS) по SSH и выполнить команду в терминале.
```
sudo su -c "bash <(curl -Ls https://raw.githubusercontent.com/LostGit77/Install-Warp/refs/heads/main/install_warp.sh)" 
```

## 🌐 WARP Proxy в 3x-ui
В панели перейти: Настройки Xray → Исходящие подключения → Создать подключение
> Переместить его вверх
```
ПРОТОКОЛ: Socks
ТЕГ.....: WARP-CLI
АДРЕС...: 127.0.0.1
ПОРТ....: 40000
```
